⚠️ Problème / Challenge

⚠️ Problem / Challenge

L'entreprise AC.Corp (15 salariés) avait besoin de séparer logiquement les services RH et Comptabilité pour des raisons de sécurité et d'organisation. Le réseau existant ne permettait pas d'isoler les flux entre départements, créant des risques d'accès non autorisés aux données sensibles.

AC.Corp company (15 employees) needed to logically separate HR and Accounting departments for security and organizational reasons. The existing network didn't allow traffic isolation between departments, creating risks of unauthorized access to sensitive data.

✅ Solution

✅ Solution

Mise en place d'une segmentation réseau complète avec configuration de 2 VLAN distincts (VLAN 10 pour RH et VLAN 20 pour COMPTA) sur un switch Cisco 2960. Configuration d'un lien trunk vers le routeur pour permettre le routage inter-VLAN (router-on-a-stick). Déploiement d'un serveur DHCP avec relais DHCP (ip helper-address) pour l'attribution automatique des adresses IP dans chaque VLAN. Plan d'adressage optimisé : 192.168.10.0/28 pour RH et 192.168.10.16/29 pour COMPTA.

Implementation of complete network segmentation with configuration of 2 distinct VLANs (VLAN 10 for HR and VLAN 20 for Accounting) on a Cisco 2960 switch. Configuration of trunk link to router for inter-VLAN routing (router-on-a-stick). Deployment of DHCP server with DHCP relay (ip helper-address) for automatic IP address assignment in each VLAN. Optimized addressing plan: 192.168.10.0/28 for HR and 192.168.10.16/29 for Accounting.

📊 Résultats

📊 Results

  • Isolation complète des flux réseau entre services
  • Sécurisation accrue de l'infrastructure
  • Facilitation de l'administration et du troubleshooting
  • Attribution automatique des IP via DHCP fonctionnelle
  • Complete network traffic isolation between departments
  • Enhanced infrastructure security
  • Simplified administration and troubleshooting
  • Functional automatic IP assignment via DHCP

⚠️ Problème / Challenge

⚠️ Problem / Challenge

Besoin de créer une maquette d'infrastructure d'entreprise complète pour simuler un environnement professionnel réaliste. L'objectif était d'implémenter une gestion centralisée des utilisateurs, des ordinateurs et des stratégies de sécurité, tout en maintenant une administration simplifiée.

Need to create a complete enterprise infrastructure mockup to simulate a realistic professional environment. The goal was to implement centralized management of users, computers and security policies, while maintaining simplified administration.

✅ Solution

✅ Solution

Déploiement complet de Windows Server 2022 avec installation et configuration des rôles Active Directory, DNS et DHCP. Structuration de l'annuaire avec création d'unités organisationnelles (OU) par service. Mise en place de stratégies de groupe (GPO) pour standardiser les configurations : déploiement automatique de logiciels, restrictions d'accès par profil, paramètres de sécurité renforcés, configuration des bureaux utilisateurs. Gestion complète des comptes utilisateurs et configuration des permissions NTFS sur les partages réseau.

Complete deployment of Windows Server 2022 with installation and configuration of Active Directory, DNS and DHCP roles. Directory structuring with creation of organizational units (OU) by department. Implementation of Group Policy Objects (GPO) to standardize configurations: automatic software deployment, profile-based access restrictions, enhanced security settings, user desktop configuration. Complete user account management and NTFS permissions configuration on network shares.

📊 Results
