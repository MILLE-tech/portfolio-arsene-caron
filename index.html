Segmentation Réseau par VLAN

Network Segmentation with VLANs

01
Cisco Switch 2960 VLAN Trunk DHCP Router-on-a-stick

⚠️ Problème

L'entreprise AC.Corp (15 salariés) avait besoin de séparer logiquement les services RH et Comptabilité pour des raisons de sécurité.

✅ Solution

2 VLAN distincts sur switch Cisco 2960, lien trunk vers routeur, DHCP avec relay. Plan d'adressage : 192.168.10.0/28 (RH) et 192.168.10.16/29 (COMPTA).

📊 Résultats

  • Isolation complète des flux réseau entre services
  • Sécurisation accrue de l'infrastructure
  • Attribution automatique des IP via DHCP

Infrastructure Virtualisée avec Active Directory

Virtualized Infrastructure with Active Directory

02
Windows Server 2022 Active Directory GPO DNS DHCP

⚠️ Problème

Besoin d'une infrastructure d'entreprise virtualisée avec gestion centralisée des utilisateurs et politiques de sécurité.

✅ Solution

Windows Server 2022 avec AD, DNS, DHCP. GPO pour déploiement automatique de logiciels et restrictions par profil.

📊 Résultats

  • Administration centralisée de plus de 20 postes clients
  • Automatisation complète du déploiement de configurations
  • Réduction significative du temps d'administration
